# FourFront Frontend Assessment

This project is a recreation of a financial membership dashboard design as part of a frontend interview assessment.

##  Getting Started

To view the dashboard, simply open `index.html` in any modern web browser.

##  Tech Stack

- **HTML5**: Semantic structure.
- **CSS3 (Bootstrap 5)**: Responsive grid system and base components.
- **Vanilla JavaScript**: Interactive elements and DOM manipulation.
- **Google Fonts**: "Inter" for modern typography.
- **Font Awesome**: Scaling vector icons.

##  Features

- **Responsive Design**: Fully functional across Mobile, Tablet, and Desktop.
- **Membership Toggles**: Interactive sections for foundation and economy memberships that reveal/hide descriptions on click.
- **Modern UI**: Dark mode profile cards, vibrant statistics grid, and a floating WhatsApp button.

##  Project Structure

- `index.html`: The main entry point containing the dashboard structure.
- `style.css`: Custom styling and overrides for the visual layout.
- `script.js`: Interactive logic for toggling descriptions.
